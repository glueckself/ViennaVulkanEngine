Detailed Description

Am ortho camera that can be used to take photos of the scene.

VECameraOrtho is derived from VECamera and can be put into the scene. It produces a projection matrix which representing the camera frustum. This class assumes a orthographic mapping. The frustum is a box and x,z are not influenced by the depth value.
