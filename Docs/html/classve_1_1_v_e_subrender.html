Detailed Description

A VESubrender instance manages and draws its entities.

Subrenderers understand the data that is attached to entities and create and manage the according Vulkan resources. The also create PSOs and use them to draw entities. Subrenderers maintain a list of entities associated with them.

Draw all associated entities.
