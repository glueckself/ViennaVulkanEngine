virtual

Draw all associated entities for the shadow pass.

The subrenderer maintains a list of all associated entities. In this function it goes through all of them and draws them. A vector is used in order to be able to parallelize this in case thousands or objects are in the list

Parameters
[in]commandBufferThe command buffer to record into all draw calls
[in]imageIndexIndex of the current swap chain image
[in]numPassThe number of the light that has been rendered
[in]pCameraPointer to the current light camera
[in]pLightPointer to the current light
[in]descriptorSetsShadowThe shadow maps to be used.
